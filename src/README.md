1. RUN ICESTORM + ICEREGISTRY

Run IceStorm and IceRegistry:

  $ ./run_icestorm

2. RUN SERVER

Run orchestrator and downloader:

    $ ./run_server

3. RUN CLIENT

    2.1. Run client to get the list of available files:

        $ ./run_client

    2.2. Run client to download a file:

        $ ./run_client <youtube-url>
